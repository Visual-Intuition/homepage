# Visual Intuition Homepage

Landing page for Visual Intuition — a startup that behavioral clones expert annotation traces from humans.

## Setup

```bash
cd src
npm install
```

## Development

```bash
cd src
npm run dev
```

Open http://localhost:3000.

## Deploy

Pushes to `main` auto-deploy via Vercel (project lives on the `nano6626` account). Live at https://visualintuition.ai.
